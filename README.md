# SafetySigns

Production-ready Next.js 14 App Router starter for a safety signs ordering platform.

## Stack

- Next.js 14 (App Router)
- TypeScript
- TailwindCSS
- React Context for cart state

## Pages

- /
- /catalog
- /cart
- /upload
- /about

## Run locally

1. npm install
2. npm run dev

## Notes

- Catalog data is stored in data/signs.json.
- Logo uploads are handled by app/api/upload/route.ts and saved in public/uploads.
- Basic i18n-ready structure is in i18n/ with English defaults.
